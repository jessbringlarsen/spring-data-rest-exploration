clubMembership

POST

Create a relation between a club and a player

Request

Headers

The request should have the Content-Type application/json

Body

Required properties
  • player: player
  • club: club
  • memberFrom: yyyy-MM-dd
Optional properties
  • memberTo : yyyy-MM-dd
Example
{
  "player": "http://localhost:8080/players/1",
  "club": "http://localhost:8080/clubs/1",
  "memberFrom": "2016-01-03"
}

Responses

201 Created

Headers
  • Location: URI of the created player
