player

POST

Create a player

Request

Headers

The request should have the Content-Type application/json

Body

Required properties
  • name: string
Optional properties
  • birthdate : yyyy-MM-dd
Example
{
  "name": "Vladimir Samsonov",
  "birthdate": "1979-03-30"
}

Responses

201 Created

Headers
  • Location: URI of the created player
